Import Data
To import your finances into this program you must have a Google Documents account. (Click here to sign up.)

Upload or create a spreadsheet with all the information to import. The first row of the spreadsheet must have the following columns: account, accountCat, date, amount, description, cleared, note. Once that is complete, tap "Continue", select your spreadsheet, then the system will import your data.
